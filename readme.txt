= Give your users the power to view and print the summary of the contact form within seconds =

**Contact Form 7 Summary and Print** is a simple-to-use add-on that gives your users the power to view the summary of your contact form and print it with the help of a "Print Summary" button.

You can also display a custom message that will be displayed after the users complete their form and click submit.

= How does it work? =

* After installation, this plugin will add a new submenu, “Summary & Print” under contact form 7 menu
* Click on that menu, configure the plugin's settings, and that's it! Yes, it's that simple.
* Users can view their form’s summary with the information they entered in the input fields during form submission. The "print summary" button allows them to print the summary form.

If you don’t want to show the "Print Summary" button on the summary page, leave the Print button’s field empty.

= Features =

* Display users their form's summary
* Allow users to print the form's summary
* Enable/Disable the "Print Summary" button on the form's summary 
* Display a custom message after form submission

== Installation ==
To add a WordPress Plugin using the built-in plugin installer:

Go to Plugins > Add New.

1. Type in the name \"Contact Form 7 Summary and Print\" in Search Plugins box
2. Find the \"Contact Form 7 Summary and Print\" Plugin you wish to install.
3. Click Install Now to install the WordPress Plugin.
4. The resulting installation screen will list the installation as successful or note any problems during the install.
If successful, click Activate Plugin to activate it, or Return to Plugin Installer for further actions.

== Frequently Asked Questions ==
= Some field's labels are not displaying on Summary Page =
If the field's labels are not appearing on Summary Page, you need to wrap that field within Contact Form 7 label

= Can I change font size and styling on Summary Page? =
There is no option for this in the plugin, but you can add your custom CSS to change the style according to your need.
